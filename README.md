# ただのtypescriptの練習

## 立ち上げ方
docker compose build  コンテナをビルドする  
docker compose up -d 裏でコンテナを起動させる  
docker compose exec app sh コンテナの中のappディレクトリに入る  
npx tsx src/main.ts typescriptをメモリ上でjsにトランスパイルして実行してくれる  
exit 対話モードを終了する  

## 学んだこと,実践したこと
shはシェルのことでコンテナの中でシェルを起動して対話できるようにする  
とりあえず基礎文法の復習の中でインターフェイスや関数をファイルに分けて実行してみた  

## 所感
dockerは正直なんかかっこいいから使ってるって感じだけど、仕組みがちょっとだけわかってきて楽しくなってきた  
関数に切り分けて副作用を少なくする（グローバル変数の使用を極力しない）すること、意味のある単位で関数化できてmain関数のロジックを薄くできた
